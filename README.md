# orskbot

Bot for Telegram channel #orsk
Invite: https://telegram.me/joinchat/B5623z5lC_jsIXaaArVhiA

### Installation
Required python3

Crearte virtualenv
```sh
python -m venv .env
```

Activate virtualenv

Windows
```sh
C:\> .env\Scripts\activate
```

Linux and OS X 
```sh
$ source .env/Scripts/activate
```

Install dependens
```sh
pip install > requirements.txt
```

### Run

Windows
```sh
bot_loop.cmd
```

Linux and OS X
```sh
$ source .env/Scripts/activate
$ python bot.py
```
